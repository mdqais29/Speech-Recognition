## OVERVIEW
This project implements a Speech-to-Text system using a pretrained Wav2Vec2 model from HuggingFace. The application accepts a short .wav audio file and transcribes spoken speech into text.

The system performs automatic speech recognition (ASR) locally without relying on external APIs.

## OBJECTIVE
- Load pretrained speech model
- Accept audio file input
- Convert speech into text
- Demonstrate audio preprocessing and inference

## TECHNOLOGIES USED
- Python
- PyTorch
- HuggingFace Transformers
- Wav2Vec2 (facebook/wav2vec2-base-960h)
- Librosa

## INSTALLATION
